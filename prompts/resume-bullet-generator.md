# Resume Bullet Generator

Create 4 ATS-friendly resume bullets from this job history.

Requirements:
- Action verb first
- Achievement-focused
- Quantify impact if possible
- Max 25 words
- Bullets only

{job_history}

Example

Input:

AI Engineering Intern. Built Python scripts, automated reports, maintained documentation, and assisted with LLM projects.

## Example Output

* Developed Python automation scripts that reduced manual effort.
* Collaborated with teams to deliver project features.
* Maintained technical documentation for workflows.
* Improved process efficiency through automation.
